🚀 Chitkara University · BE 2025
Train Smarter,
Achieve More
APEX is a complete fitness tracking platform combining workout logging, smart nutrition analysis powered by real food APIs, body weight monitoring, and an intelligent coach.
🎯
Our Objective

APEX was built to make elite fitness tracking accessible to everyone. By keeping workouts, nutrition, and insights in one place, you identify patterns, break plateaus, and hit goals faster — without needing a personal trainer.

⚡
What Makes APEX Different
AI Nutrition Engine — Type "2 eggs and toast" and macros auto-fill instantly via Claude AI.
Per-User Data — Each user has isolated data via a unique browser ID.
Live Visitor Count — Real-time global visitor counter via CounterAPI.
Privacy First — No account, no signup. All data stays in your browser.
Core Features
📊
Smart Dashboard
Live stats for calories, workouts, protein, and body weight updated as you log.
🏋️
Workout Logger
Log any exercise with sets, reps, weight. Auto-estimates calories, tracks PRs.
🥗
Auto-Nutrition
Powered by Claude AI — type natural language and get instant macro breakdowns.
📈
Progress Analytics
Weight trend chart, personal best leaderboard, and session history.
🤖
Smart Coach
Rule-based coach that reads your real data and gives personalised advice.
👥
Visitor Counter
Real-time global visit count via CounterAPI. Each user gets isolated data.
